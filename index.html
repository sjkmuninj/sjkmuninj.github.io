发表于 2021年06月16 分类于 前端 ， JavaScript

Http状态码

http状态码有5种,分别是

  1. (1**)消息,服务器收到请求,需要执行请求者继续执行操作
  2. (2**)成功,操作被成功接收并处理
  3. (3**)重定向,需要进一步操作以完成请求
  4. (4**)客户端错误,请求包含语法错误或无法完成请求
  5. (5**)服务器错误,服务器在处理请求过程中发生了错误
